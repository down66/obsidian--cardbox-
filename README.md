# obsidian--cardbox-
cardbox 是一个为 Obsidian 设计的卡片盒插件，用来把零散想法、素材片段和灵感卡片从普通长笔记中分离出来管理。 它提供独立的卡片工作台：左侧可以像编辑普通 Obsidian 笔记一样编辑卡片，右侧以网格方式预览所有卡片，并支持卡片盒分类、标签筛选、排序、图片预览、批量删除和视图大小调整。卡片本身以 Markdown 文件保存，可以插入到普通笔记中，并在 Info 面板中查看创建时间、字数、所属卡片盒、标签和反向链接。 适合想在 Obsidian 中建立类似 Heptabase 风格卡片系统、视觉化整理碎片信息，同时保留 Markdown 和双链生态兼容性的用户。

english:`cardbox` is an Obsidian plugin designed to separate fragmented ideas and material snippets from long-form notes. It provides a dedicated card workspace where cards can be edited like normal Obsidian notes on the left and previewed in a grid on the right. It supports card boxes, tag filtering, sorting, image previews, bulk actions, adjustable view sizes, and rich Info panels with metadata and backlinks. Cards are stored as Markdown files, can be embedded into regular notes, and stay compatible with Obsidian’s linking and Markdown ecosystem.

There is currently no English version; only the Chinese version is available.
